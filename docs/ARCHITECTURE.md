# DIOS Architecture

Status: `PRE-ALPHA`

```text
source package
→ project and revision context
→ document and sheet registry
→ extraction
→ object graph
→ measurement plan
→ quantity engine
→ validation
→ review
→ exports
```

## Shared core

- project context
- roles and approvals
- source and artifact registry
- revision tracking
- workflow execution
- calculation interfaces
- evidence and audit
- conflict and review queues

## DIOS layer

- PDF, raster, vector, DXF and IFC adapters
- sheet, revision, unit and scale extraction
- cross-sheet relations
- room, wall, opening and system schemas
- geometry and quantity methods
- drawing validation
- marked-up evidence

Canonical data remains structured and versioned. Spreadsheets and PDFs are generated views.
